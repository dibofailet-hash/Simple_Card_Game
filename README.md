## 📌 О проекте

**Programming Quiz App** — это десктопное приложение с графическим интерфейсом,
реализующее интерактивную викторину по основам программирования.
Пользователь отвечает на вопросы, выбирая один из четырёх вариантов ответа,
получает мгновенную обратную связь и переходит к следующему вопросу.

---

## 🖥️ Интерфейс

| Элемент | Описание |
|---|---|
| Текст вопроса | Отображается в верхней части окна |
| 4 радиокнопки | Варианты ответа, разбитые на 2 колонки |
| Кнопка «Проверить» | Проверяет выбранный ответ |
| Метка результата | Показывает правильность ответа |
| Кнопка «Следующий вопрос» | Появляется после проверки |

---

## ✨ Функциональность

- ✅ **9 вопросов** по теме основ программирования
- 🔘 Выбор ответа через **радиокнопки** (только один вариант)
- 🔍 Мгновенная **проверка ответа** с текстовым сообщением
- 🙈 **Скрытие вариантов** после ответа для чистоты интерфейса
- 🔒 **Блокировка кнопки** по завершении всех вопросов
- 🔄 Последовательный **переход между вопросами**

---

## 🗂️ Темы вопросов

- Языки программирования (JavaScript, Java, Python...)
- Базовые конструкции: переменные, циклы, условия, функции
- Типы данных: String, Boolean, Integer, Float
- Структуры данных: массивы
- Инструменты разработчика: IDE

---

## 🛠️ Технологии

| Технология | Применение |
|---|---|
| **Python 3.9** | Язык разработки |
| **PyQt5** | Графический интерфейс (GUI) |
| `QWidget` | Базовое окно приложения |
| `QGroupBox` | Группировка радиокнопок |
| `QRadioButton` | Кнопки выбора ответа |
| `QButtonGroup` | Обеспечение единственного выбора (`setExclusive`) |
| `QVBoxLayout` / `QHBoxLayout` | Вертикальная и горизонтальная компоновка |
| `QLabel` | Отображение вопроса и результата |
| `QPushButton` | Кнопки «Проверить» / «Следующий вопрос» |
